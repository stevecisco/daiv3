# Layer Interface Specifications - DAIv3

## Purpose

This document provides detailed C# interface specifications that define the contracts between layers. All implementations must strictly conform to these interfaces and their documented behaviors.

---

## PERSISTENCE LAYER INTERFACES

### Namespace: `Daiv3.Persistence.Interfaces`

#### IEntity
Base interface for all persistent entities.

```csharp
/// <summary>
/// Base interface for entities stored in the persistence layer.
/// All entities must have unique identity and audit timestamps.
/// </summary>
public interface IEntity
{
    /// <summary>Globally unique identifier for this entity.</summary>
    Guid Id { get; }
    
    /// <summary>UTC timestamp when entity was created.</summary>
    DateTimeOffset CreatedAt { get; }
    
    /// <summary>UTC timestamp when entity was last modified (null if never modified).</summary>
    DateTimeOffset? UpdatedAt { get; }
}
```

#### IRepository<T>
Generic repository interface for entity persistence.

```csharp
/// <summary>
/// Generic repository for CRUD operations on entities.
/// </summary>
/// <typeparam name="T">Entity type implementing IEntity</typeparam>
/// <remarks>
/// - All operations are async to prevent blocking
/// - Transactions are handled at the service layer
/// - Repository does not perform authorization checks
/// </remarks>
public interface IRepository<T> where T : IEntity
{
    /// <summary>
    /// Retrieves an entity by its unique identifier.
    /// </summary>
    /// <param name="id">Entity ID</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Entity or null if not found</returns>
    Task<T?> GetByIdAsync(Guid id, CancellationToken ct = default);
    
    /// <summary>
    /// Retrieves all entities of this type.
    /// </summary>
    /// <remarks>
    /// For large result sets, pagination should be implemented via overload.
    /// </remarks>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Read-only collection (may be empty)</returns>
    Task<IReadOnlyList<T>> GetAllAsync(CancellationToken ct = default);
    
    /// <summary>
    /// Retrieves entities matching a predicate.
    /// </summary>
    /// <param name="predicate">Filter expression</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Matching entities (may be empty)</returns>
    Task<IReadOnlyList<T>> GetByPredicateAsync(
        Func<T, bool> predicate, 
        CancellationToken ct = default);
    
    /// <summary>
    /// Adds a new entity to storage.
    /// </summary>
    /// <remarks>
    /// - Entity ID should be generated by the caller (typically Guid.NewGuid())
    /// - CreatedAt is set by the repository
    /// - UpdatedAt is initially null
    /// </remarks>
    /// <param name="entity">Entity to add</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Added entity with ID and timestamps populated</returns>
    /// <exception cref="ArgumentException">If entity ID already exists</exception>
    /// <exception cref="InvalidOperationException">If storage operation fails</exception>
    Task<T> AddAsync(T entity, CancellationToken ct = default);
    
    /// <summary>
    /// Updates an existing entity.
    /// </summary>
    /// <remarks>
    /// - ID must match an existing entity
    /// - UpdatedAt is set to current UTC time
    /// </remarks>
    /// <param name="entity">Entity with updated values</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Updated entity</returns>
    /// <exception cref="InvalidOperationException">If entity ID does not exist</exception>
    Task<T> UpdateAsync(T entity, CancellationToken ct = default);
    
    /// <summary>
    /// Deletes an entity by ID.
    /// </summary>
    /// <param name="id">ID of entity to delete</param>
    /// <param name="ct">Cancellation token</param>
    /// <exception cref="InvalidOperationException">If entity does not exist</exception>
    Task DeleteAsync(Guid id, CancellationToken ct = default);
    
    /// <summary>
    /// Counts entities matching an optional predicate.
    /// </summary>
    /// <param name="predicate">Optional filter (null = all)</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Number of matching entities</returns>
    Task<int> CountAsync(
        Func<T, bool> predicate = null, 
        CancellationToken ct = default);
}
```

#### IDatabaseFactory
Factory for creating and managing database connections.

```csharp
/// <summary>
/// Factory for creating SQLite database connections and managing migrations.
/// </summary>
/// <remarks>
/// Implementations should:
/// - Use connection pooling for performance
/// - Ensure migrations run automatically on first connection
/// - Handle concurrent access safely
/// </remarks>
public interface IDatabaseFactory
{
    /// <summary>
    /// Creates a new database connection.
    /// </summary>
    /// <remarks>
    /// - Connection is opened immediately
    /// - Caller must dispose the connection
    /// - Should use connection pooling internally
    /// </remarks>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Open SqliteConnection</returns>
    /// <exception cref="InvalidOperationException">If database cannot be accessed</exception>
    Task<DbConnection> CreateConnectionAsync(CancellationToken ct = default);
    
    /// <summary>
    /// Applies all pending migrations to the database.
    /// </summary>
    /// <remarks>
    /// - Safe to call multiple times (idempotent)
    /// - Should create schema on first call
    /// - Should update schema on subsequent calls
    /// </remarks>
    /// <param name="ct">Cancellation token</param>
    /// <exception cref="InvalidOperationException">If migrations fail</exception>
    Task ApplyMigrationsAsync(CancellationToken ct = default);
    
    /// <summary>
    /// Generates an encrypted connection string for the database.
    /// </summary>
    /// <returns>Connection string</returns>
    string GetConnectionString();
}
```

### Namespace: `Daiv3.Infrastructure.Shared.Hardware`

#### IHardwareDetectionProvider
Hardware capability detection.

```csharp
/// <summary>
/// Detects available hardware capabilities for execution.
/// </summary>
/// <remarks>
/// - Should cache results after first detection
/// - Detection should run on a separate thread to avoid blocking
/// - Results are immutable after initial detection
/// </remarks>
public interface IHardwareDetectionProvider
{
    /// <summary>
    /// Detects all hardware capabilities once.
    /// </summary>
    /// <returns>Snapshot of current hardware capabilities</returns>
    HardwareCapabilities DetectCapabilities();
    
    /// <summary>Whether system has a dedicated NPU.</summary>
    bool HasNpu { get; }
    
    /// <summary>Whether system has a dedicated GPU.</summary>
    bool HasGpu { get; }
    
    /// <summary>CPU/processor name as reported by OS.</summary>
    string CpuName { get; }
    
    /// <summary>Operating system and version string.</summary>
    string OsVersion { get; }
    
    /// <summary>
    /// Gets the preferred execution provider for this hardware.
    /// </summary>
    /// <remarks>
    /// Priority: NPU > GPU > CPU
    /// </remarks>
    ExecutionProvider GetPreferredProvider();
}

/// <summary>
/// Immutable snapshot of hardware capabilities.
/// </summary>
public class HardwareCapabilities
{
    public bool HasNpu { get; init; }
    public bool HasGpu { get; init; }
    public string CpuName { get; init; }
    public string OsVersion { get; init; }
    public long AvailableMemoryBytes { get; init; }
    public int LogicalCoreCount { get; init; }
    public DateTimeOffset DetectedAt { get; init; }
}

/// <summary>
/// Execution provider preference.
/// </summary>
public enum ExecutionProvider
{
    /// <summary>Neural Processing Unit (primary target for Copilot+ devices)</summary>
    Npu = 0,
    
    /// <summary>GPU fallback</summary>
    Gpu = 1,
    
    /// <summary>CPU fallback (SIMD optimized)</summary>
    Cpu = 2,
    
    /// <summary>Auto-detect based on capabilities</summary>
    Auto = 3
}
```

## KNOWLEDGE LAYER INTERFACES

### Namespace: `Daiv3.Knowledge.DocProc.Interfaces`

#### IDocumentProcessor
Multi-format document extraction.

```csharp
/// <summary>
/// Extracts text and metadata from documents in various formats.
/// </summary>
/// <remarks>
/// - Should support PDF, DOCX, HTML, Markdown, TXT, and source code files
/// - Must handle encoding detection and normalization
/// - Should preserve structural information (headers, lists, etc.)
/// </remarks>
public interface IDocumentProcessor
{
    /// <summary>
    /// Processes a document file and extracts text.
    /// </summary>
    /// <remarks>
    /// - File must exist and be readable
    /// - Encoding is auto-detected
    /// - Large files (>100MB) should be handled gracefully
    /// </remarks>
    /// <param name="filePath">Full path to document file</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Extracted text and metadata</returns>
    /// <exception cref="FileNotFoundException">If file does not exist</exception>
    /// <exception cref="InvalidOperationException">If format cannot be parsed</exception>
    Task<ProcessedDocument> ProcessAsync(string filePath, CancellationToken ct = default);
    
    /// <summary>
    /// Checks if this processor can handle the file format.
    /// </summary>
    /// <remarks>
    /// Check should be based on file extension (case-insensitive).
    /// </remarks>
    /// <param name="extension">File extension (e.g., ".pdf", ".docx")</param>
    /// <returns>True if processor can handle this format</returns>
    bool SupportsFormat(string extension);
}

/// <summary>
/// Result of document processing.
/// </summary>
public class ProcessedDocument
{
    /// <summary>Extracted text content.</summary>
    public string Content { get; set; }
    
    /// <summary>
    /// Metadata extracted from document.
    /// Examples: Author, Title, CreationDate, PageCount, etc.
    /// </summary>
    public Dictionary<string, string> Metadata { get; set; }
    
    /// <summary>Detected language code (e.g., "en", "fr").</summary>
    public string DetectedLanguage { get; set; }
    
    /// <summary>Original file path.</summary>
    public string SourcePath { get; set; }
}
```

#### ITextChunker
Document chunking into token-bounded segments.

```csharp
/// <summary>
/// Splits long text into chunks with configurable token limits.
/// </summary>
/// <remarks>
/// - Uses ITokenizerProvider for token counting
/// - Preserves sentence boundaries when possible
/// - Overlapping chunks enable context continuity
/// </remarks>
public interface ITextChunker
{
    /// <summary>
    /// Splits text into overlapping chunks.
    /// </summary>
    /// <remarks>
    /// - Creates chunks of approximately targetTokenCount tokens
    /// - Uses overlapTokens for context preservation
    /// - Attempts to break at sentence boundaries
    /// - If single sentence exceeds target, includes whole sentence
    /// </remarks>
    /// <param name="text">Text to chunk</param>
    /// <param name="targetTokenCount">Target tokens per chunk (default: 400)</param>
    /// <param name="overlapTokens">Tokens to overlap between chunks (default: 50)</param>
    /// <returns>List of chunks ordered by appearance</returns>
    List<TextChunk> ChunkText(
        string text, 
        int targetTokenCount = 400, 
        int overlapTokens = 50);
}

/// <summary>
/// Represents a single text chunk.
/// </summary>
public class TextChunk
{
    /// <summary>Chunk content.</summary>
    public string Content { get; set; }
    
    /// <summary>Character position in original text where chunk starts.</summary>
    public int StartPosition { get; set; }
    
    /// <summary>Character position in original text where chunk ends.</summary>
    public int EndPosition { get; set; }
    
    /// <summary>Approximate token count.</summary>
    public int TokenCount { get; set; }
    
    /// <summary>Chunk sequence number (0-based).</summary>
    public int ChunkIndex { get; set; }
}
```

### Namespace: `Daiv3.Knowledge.Embedding.Interfaces`

#### IEmbeddingService
Embedding generation via ONNX Runtime.

```csharp
/// <summary>
/// Generates embedding vectors from text using ONNX Runtime.
/// </summary>
/// <remarks>
/// - Supports hardware acceleration (NPU > GPU > CPU)
/// - Produces float32 vectors normalized by dimensionality
/// - Batch operations are preferred for efficiency
/// - All operations are async and cancellable
/// </remarks>
public interface IEmbeddingService
{
    /// <summary>
    /// Generates a single embedding vector for text.
    /// </summary>
    /// <remarks>
    /// - Text should be < 384 tokens (varies by model)
    /// - Very long text is truncated silently
    /// - Returns vector with values in [-1, 1] range
    /// </remarks>
    /// <param name="text">Text to embed (non-empty)</param>
    /// <param name="dimensionality">Target dimensions (384 or 768)</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Float32 embedding vector of specified dimension</returns>
    /// <exception cref="ArgumentException">If text is empty or dimensionality unsupported</exception>
    /// <exception cref="InvalidOperationException">If ONNX inference fails</exception>
    Task<float[]> GenerateEmbeddingAsync(
        string text, 
        int dimensionality, 
        CancellationToken ct = default);
    
    /// <summary>
    /// Generates embeddings for multiple texts in a batch.
    /// </summary>
    /// <remarks>
    /// - More efficient than calling GenerateEmbeddingAsync repeatedly
    /// - Processes texts in groups to optimize hardware usage
    /// - Preserves order: results[i] matches texts[i]
    /// </remarks>
    /// <param name="texts">Texts to embed (all non-empty)</param>
    /// <param name="dimensionality">Target dimensions (384 or 768)</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>List of embedding vectors</returns>
    /// <exception cref="ArgumentException">If texts is empty or any text is empty</exception>
    Task<List<float[]>> GenerateBatchEmbeddingsAsync(
        List<string> texts, 
        int dimensionality, 
        CancellationToken ct = default);
    
    /// <summary>Name of the embedding model (e.g., "all-MiniLM-L6-v2").</summary>
    string ModelName { get; }
    
    /// <summary>Default dimensionality for this model (384 or 768).</summary>
    int DefaultDimensions { get; }
    
    /// <summary>
    /// Gets the maximum token length this model can process.
    /// </summary>
    int MaxTokenLength { get; }
}
```

#### IVectorSimilarityService
Vector math operations.

```csharp
/// <summary>
/// Computes vector similarity using SIMD-optimized operations.
/// </summary>
/// <remarks>
/// - Uses TensorPrimitives for CPU optimization
/// - Cosine similarity is the standard metric
/// - All operations work with float32 vectors
/// </remarks>
public interface IVectorSimilarityService
{
    /// <summary>
    /// Computes cosine similarity between two vectors.
    /// </summary>
    /// <remarks>
    /// - Both vectors must be same dimension
    /// - Result range: [0, 1] for normalized vectors
    /// - Result 1.0 = identical vectors
    /// - Result 0.0 = orthogonal vectors
    /// </remarks>
    /// <param name="vector1">First vector</param>
    /// <param name="vector2">Second vector</param>
    /// <returns>Cosine similarity score</returns>
    /// <exception cref="ArgumentException">If vectors have different dimensions</exception>
    float ComputeCosineSimilarity(float[] vector1, float[] vector2);
    
    /// <summary>
    /// Computes similarity between one query vector and multiple document vectors.
    /// </summary>
    /// <remarks>
    /// - Returns top-k most similar vectors
    /// - Results sorted by similarity (descending)
    /// - More efficient than calling ComputeCosineSimilarity repeatedly
    /// </remarks>
    /// <param name="queryVector">Query embedding</param>
    /// <param name="documentVectors">Document embeddings to compare</param>
    /// <param name="topK">Number of top results to return</param>
    /// <returns>List of (index, similarity) tuples sorted by similarity</returns>
    /// <exception cref="ArgumentException">If vectors have inconsistent dimensions</exception>
    List<(int Index, float Similarity)> ComputeBatchSimilarities(
        float[] queryVector, 
        List<float[]> documentVectors, 
        int topK = 10);
}
```

#### IKnowledgeIndex
Two-tier semantic search interface.

```csharp
/// <summary>
/// Implements two-tier vector indexing for semantic search.
/// </summary>
/// <remarks>
/// - Tier 1: Topic level (one vector per document)
/// - Tier 2: Chunk level (multiple vectors per document)
/// - Tier 1 is searched first for broad matching
/// - Tier 2 is searched only on Tier 1 results for precision
/// </remarks>
public interface IKnowledgeIndex
{
    /// <summary>
    /// Adds a new document and generates its embeddings.
    /// </summary>
    /// <remarks>
    /// - Automatically chunks content and generates Tier 1/2 embeddings
    /// - Overwrites if document already exists (by path)
    /// - Operations are atomic (all or nothing)
    /// </remarks>
    /// <param name="filePath">Document file path for identification</param>
    /// <param name="content">Document text content</param>
    /// <param name="metadata">Document metadata (author, date, etc.)</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Document ID</returns>
    /// <exception cref="InvalidOperationException">If embedding generation fails</exception>
    Task<Guid> AddDocumentAsync(
        string filePath, 
        string content, 
        Dictionary<string, string> metadata, 
        CancellationToken ct = default);
    
    /// <summary>
    /// Searches Tier 1 (topic-level) index for similar documents.
    /// </summary>
    /// <remarks>
    /// - Fast coarse search across all documents
    /// - Should complete in <10ms for ~10k documents
    /// - Returns document candidates for Tier 2 refinement
    /// </remarks>
    /// <param name="query">User search query</param>
    /// <param name="topK">Number of results (default: 10)</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Document IDs with similarity scores (sorted descending)</returns>
    Task<List<(Guid DocumentId, float Score)>> SearchTier1Async(
        string query, 
        int topK = 10, 
        CancellationToken ct = default);
    
    /// <summary>
    /// Searches Tier 2 (chunk-level) index within document candidates.
    /// </summary>
    /// <remarks>
    /// - Fine-grained search for specific content
    /// - Only called on Tier 1 results for efficiency
    /// - Can return multiple chunks per document
    /// </remarks>
    /// <param name="documentCandidates">Document IDs from Tier 1 search</param>
    /// <param name="query">User search query</param>
    /// <param name="topK">Number of chunk results (default: 20)</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Chunk IDs with similarity scores (sorted descending)</returns>
    Task<List<(Guid ChunkId, float Score)>> SearchTier2Async(
        List<Guid> documentCandidates, 
        string query, 
        int topK = 20, 
        CancellationToken ct = default);
    
    /// <summary>
    /// Combined Tier 1 + Tier 2 search in one call.
    /// </summary>
    /// <remarks>
    /// Convenience method that calls SearchTier1Async then SearchTier2Async.
    /// </remarks>
    /// <param name="query">User search query</param>
    /// <param name="tier1TopK">Tier 1 candidate count</param>
    /// <param name="tier2TopK">Tier 2 result count</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Final chunk results</returns>
    Task<List<(Guid ChunkId, float Score)>> SearchAsync(
        string query, 
        int tier1TopK = 10, 
        int tier2TopK = 20, 
        CancellationToken ct = default);
    
    /// <summary>
    /// Removes a document and all its embeddings.
    /// </summary>
    /// <remarks>
    /// - Deletes document and all associated chunks from storage
    /// - Safe to call if document doesn't exist
    /// </remarks>
    /// <param name="documentId">Document to remove</param>
    /// <param name="ct">Cancellation token</param>
    Task RemoveDocumentAsync(Guid documentId, CancellationToken ct = default);
    
    /// <summary>
    /// Gets total count of indexed documents.
    /// </summary>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Document count</returns>
    Task<int> GetDocumentCountAsync(CancellationToken ct = default);
}
```

## MODEL EXECUTION LAYER INTERFACES

### Namespace: `Daiv3.ModelExecution.Interfaces`

#### IModelQueue
Priority-based request queue.

```csharp
/// <summary>
/// Manages model execution requests with priority-based scheduling.
/// </summary>
/// <remarks>
/// - P0: Immediate (preempt everything)
/// - P1: Normal (batch before switching)
/// - P2: Background (drain before switching)
/// - Only one model loaded at a time
/// </remarks>
public interface IModelQueue
{
    /// <summary>
    /// Enqueues an execution request.
    /// </summary>
    /// <remarks>
    /// - Immediate return with request ID
    /// - Request is executed asynchronously
    /// - Use GetStatusAsync to poll results
    /// </remarks>
    /// <param name="request">Execution request</param>
    /// <param name="priority">Execution priority (default: Normal)</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Request ID for tracking</returns>
    Task<Guid> EnqueueAsync(
        ExecutionRequest request, 
        ExecutionPriority priority = ExecutionPriority.Normal, 
        CancellationToken ct = default);
    
    /// <summary>
    /// Processes a request and waits for completion.
    /// </summary>
    /// <remarks>
    /// - Blocks until request completes or timeout
    /// - Used when immediate result needed
    /// - Check IsCompleted before accessing Result
    /// </remarks>
    /// <param name="requestId">Request ID from EnqueueAsync</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Execution result</returns>
    /// <exception cref="InvalidOperationException">If request not found</exception>
    /// <exception cref="OperationCanceledException">If cancelled</exception>
    Task<ExecutionResult> ProcessAsync(Guid requestId, CancellationToken ct = default);
    
    /// <summary>
    /// Gets status of an enqueued request.
    /// </summary>
    /// <param name="requestId">Request ID</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Current status snapshot</returns>
    Task<ExecutionRequestStatus> GetStatusAsync(Guid requestId, CancellationToken ct = default);
    
    /// <summary>
    /// Gets count of pending requests by priority.
    /// </summary>
    /// <returns>Status summary</returns>
    Task<QueueStatus> GetQueueStatusAsync();
}

/// <summary>Execution request priority level.</summary>
public enum ExecutionPriority
{
    /// <summary>Immediate: Preempt current model, switch if needed</summary>
    Immediate = 0,
    
    /// <summary>Normal: Batch with current model, then execute</summary>
    Normal = 1,
    
    /// <summary>Background: Batch and drain before model switch</summary>
    Background = 2
}

/// <summary>User's inference request.</summary>
public class ExecutionRequest
{
    /// <summary>Unique request ID (generated by producer).</summary>
    public Guid Id { get; set; }
    
    /// <summary>Task type for model selection (e.g., "chat", "summarize").</summary>
    public string TaskType { get; set; }
    
    /// <summary>Request content (prompt, question, etc.).</summary>
    public string Content { get; set; }
    
    /// <summary>Contextual information (document references, memory, etc.).</summary>
    public Dictionary<string, string> Context { get; set; }
    
    /// <summary>When request was created.</summary>
    public DateTimeOffset CreatedAt { get; set; }
}

/// <summary>Result of request execution.</summary>
public class ExecutionResult
{
    /// <summary>Corresponding request ID.</summary>
    public Guid RequestId { get; set; }
    
    /// <summary>Model output text.</summary>
    public string Content { get; set; }
    
    /// <summary>Execution status.</summary>
    public ExecutionStatus Status { get; set; }
    
    /// <summary>When execution completed.</summary>
    public DateTimeOffset CompletedAt { get; set; }
    
    /// <summary>Error message if Status == Failed.</summary>
    public string ErrorMessage { get; set; }
    
    /// <summary>Token usage statistics.</summary>
    public TokenUsage TokenUsage { get; set; }
}

/// <summary>Execution status.</summary>
public enum ExecutionStatus
{
    Queued,
    Processing,
    Completed,
    Failed,
    Cancelled
}

/// <summary>Token usage in a request.</summary>
public class TokenUsage
{
    public int InputTokens { get; set; }
    public int OutputTokens { get; set; }
    public int TotalTokens => InputTokens + OutputTokens;
}

/// <summary>Request status snapshot.</summary>
public class ExecutionRequestStatus
{
    public Guid RequestId { get; set; }
    public ExecutionStatus Status { get; set; }
    public int QueuePosition { get; set; }
    public ExecutionResult Result { get; set; }
}

/// <summary>Snapshot of queue state.</summary>
public class QueueStatus
{
    public int ImmediateCount { get; set; }  // P0
    public int NormalCount { get; set; }     // P1
    public int BackgroundCount { get; set; } // P2
    public string CurrentModelId { get; set; }
    public DateTimeOffset LastModelSwitch { get; set; }
}
```

#### IModelManagementService
Foundry Local model lifecycle.

```csharp
/// <summary>
/// Manages model downloading, loading, and lifecycle.
/// </summary>
/// <remarks>
/// - Integrates with Foundry Local service catalog
/// - Models are cached locally for offline access
/// - Single model loaded at a time (enforced by queue)
/// </remarks>
public interface IModelManagementService
{
    /// <summary>
    /// Lists all models available from Foundry service.
    /// </summary>
    /// <remarks>
    /// - Does not download models
    /// - Returns metadata for user selection
    /// </remarks>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Available models with metadata</returns>
    Task<List<AvailableModel>> ListAvailableModelsAsync(CancellationToken ct = default);
    
    /// <summary>
    /// Downloads a model from Foundry to local cache.
    /// </summary>
    /// <remarks>
    /// - Idempotent: safe to call if model exists
    /// - May take several minutes for large models
    /// - Uses shared Foundry cache directory
    /// </remarks>
    /// <param name="modelId">Model to download</param>
    /// <param name="progress">Progress callback (optional)</param>
    /// <param name="ct">Cancellation token</param>
    /// <exception cref="HttpRequestException">If download fails</exception>
    Task DownloadModelAsync(
        string modelId, 
        IProgress<DownloadProgress> progress = null, 
        CancellationToken ct = default);
    
    /// <summary>
    /// Lists all models in local cache.
    /// </summary>
    /// <remarks>
    /// - Includes models partially downloaded
    /// - Check model files before assuming ready
    /// </remarks>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Cached model IDs</returns>
    Task<List<string>> GetCachedModelsAsync(CancellationToken ct = default);
    
    /// <summary>
    /// Deletes a cached model to free space.
    /// </summary>
    /// <remarks>
    /// - Safe if model not cached
    /// - Cannot delete loaded model
    /// </remarks>
    /// <param name="modelId">Model to delete</param>
    /// <param name="ct">Cancellation token</param>
    Task DeleteModelAsync(string modelId, CancellationToken ct = default);
    
    /// <summary>
    /// Loads a cached model into memory.
    /// </summary>
    /// <remarks>
    /// - Unloads previous model automatically
    /// - Calls IDisposable on unloaded model
    /// - May take several seconds for large models
    /// </remarks>
    /// <typeparam name="T">Model type returned (varies by provider)</typeparam>
    /// <param name="modelId">Model to load</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Loaded model instance</returns>
    /// <exception cref="FileNotFoundException">If model not cached</exception>
    /// <exception cref="InvalidOperationException">If load fails</exception>
    Task<T> LoadModelAsync<T>(string modelId, CancellationToken ct = default) where T : class;
    
    /// <summary>
    /// Unloads the currently loaded model.
    /// </summary>
    /// <remarks>
    /// - Frees GPU/CPU memory
    /// - Safe if no model loaded
    /// </remarks>
    /// <param name="modelId">Model to unload (must be current)</param>
    /// <param name="ct">Cancellation token</param>
    Task UnloadModelAsync(string modelId, CancellationToken ct = default);
    
    /// <summary>Gets the currently loaded model ID (or null).</summary>
    Task<string> GetLoadedModelAsync();
}
```

## ORCHESTRATION LAYER INTERFACES

### Namespace: `Daiv3.Orchestration.Interfaces`

#### ITaskOrchestrator
High-level task coordination.

```csharp
/// <summary>
/// Orchestrates complex multi-step tasks from user requests.
/// </summary>
/// <remarks>
/// - Resolves user intent
/// - Decomposes into sub-tasks
/// - Manages dependencies
/// - Coordinates execution
/// </remarks>
public interface ITaskOrchestrator
{
    /// <summary>
    /// Executes a user request end-to-end.
    /// </summary>
    /// <remarks>
    /// - Blocks until all tasks complete or timeout
    /// - Creates a session for tracking
    /// - May spawn multiple sub-tasks
    /// </remarks>
    /// <param name="request">User request</param>
    /// <param name="ct">Cancellation token for whole operation</param>
    /// <returns>Orchestration result with all task outputs</returns>
    Task<OrchestrationResult> ExecuteAsync(
        UserRequest request, 
        CancellationToken ct = default);
    
    /// <summary>
    /// Decomposes a user input into individual tasks.
    /// </summary>
    /// <remarks>
    /// - Does not execute, only plans
    /// - Validates dependencies
    /// - Assigns execution order
    /// </remarks>
    /// <param name="userInput">User request text</param>
    /// <param name="ct">Cancellation token</param>
    /// <returns>Planned tasks with dependencies</returns>
    Task<List<ResolvedTask>> ResolveIntentAsync(
        string userInput, 
        CancellationToken ct = default);
}

/// <summary>User's input request.</summary>
public class UserRequest
{
    /// <summary>User input text.</summary>
    public string Input { get; set; }
    
    /// <summary>Project scope (if relevant).</summary>
    public Guid ProjectId { get; set; }
    
    /// <summary>Session context and memory.</summary>
    public Dictionary<string, string> Context { get; set; }
}

/// <summary>Resolved task from intent analysis.</summary>
public class ResolvedTask
{
    /// <summary>Task type for model selection.</summary>
    public string TaskType { get; set; }
    
    /// <summary>Task-specific parameters.</summary>
    public Dictionary<string, string> Parameters { get; set; }
    
    /// <summary>Order of execution (0-based).</summary>
    public int ExecutionOrder { get; set; }
    
    /// <summary>IDs of tasks that must complete first.</summary>
    public List<Guid> Dependencies { get; set; }
}

/// <summary>Result of end-to-end orchestration.</summary>
public class OrchestrationResult
{
    /// <summary>Session ID for tracking this request.</summary>
    public Guid SessionId { get; set; }
    
    /// <summary>All completed task results.</summary>
    public List<ExecutionResult> TaskResults { get; set; } = new();
    
    /// <summary>Overall success indicator.</summary>
    public bool Success { get; set; }
    
    /// <summary>Error message if failed.</summary>
    public string ErrorMessage { get; set; }
    
    /// <summary>When orchestration started.</summary>
    public DateTimeOffset StartedAt { get; set; }
    
    /// <summary>When orchestration completed.</summary>
    public DateTimeOffset CompletedAt { get; set; }
    
    /// <summary>Total duration.</summary>
    public TimeSpan Duration => CompletedAt - StartedAt;
}
```

## Summary

These interface specifications form the contracts between layers. Implementations must:

1. **Honor the Interfaces Exactly:** No modifications to signatures
2. **Document Behavior:** Implementations should document their specific behavior
3. **Handle Errors Gracefully:** Use the specified exceptions
4. **Support Async/Cancellation:** All I/O operations must be async and cancellable
5. **Validate Inputs:** Check preconditions and throw specified exceptions
6. **Log Operations:** Use ILogger throughout
